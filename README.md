# Jediswap

Clone of Uniswap V2 to Cairo. AMM for StarkNet.

## Testing and Development

### Dependencies

* [python3](https://www.python.org/downloads/release/python-3910/)
* [nile](https://github.com/OpenZeppelin/nile)


### Compile Contracts
```
nile compile
```

### Run Tests
```
pytest -s -v
```
